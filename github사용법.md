## git Brunch
1. 브런츠란?

브랜치는 Git에서 작업을 분리하기 위해 사용하는 기능입니다. 새로운 브랜치를 생성하면 기존 코드를 유지하면서도 독립적으로 작업을 진행할 수 있습니다. 작업이 완료되면 병합(merge)을 통해 메인 브랜치에 반영합니다.



- 1단계: 현재 브랜치 확인
    현재 작업 중인 브랜치를 확인하려면 다음 명령어를 사용합니다:

    ```
    git branch
    ```
    출력예시
    ```
    * main
    develop
    ```
